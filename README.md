# Silpo Online Order

Навчальний проєкт , який моделює онлайн-замовлення магазину «Сільпо».

## Можливості
- Додавання товару в замовлення
- Видалення товару із замовлення
- Зміна кількості товару
- Встановлення статусу замовлення:
  - `виконано`
  - `скасовано`
  - `оплачено`
  - `неоплачено`
- Встановлення дати доставки
- Обчислення загальної суми замовлення
- Збереження замовлення у файл
- Зчитування замовлення з файлу

## Використані інструменти STL
- `vector` — зберігання списку товарів
- `string` — робота з текстовими полями
- `fstream` — робота з файлами

## Структура проєкту
- `SilpoOrder.h` — опис структур, enum та класу
- `SilpoOrder.cpp` — реалізація методів класу
- `silpo_main.cpp` — консольне меню для роботи з замовленням

## Компіляція і запуск
Перейти в папку `silpo` і виконати:

```bash
g++ -std=c++17 silpo_main.cpp SilpoOrder.cpp -o silpo_app
./silpo_app
```

## Формат файлу збереження
Після `saveToFile()` дані записуються у такому форматі:

1. Рядок 1: статус замовлення (текст)
2. Рядок 2: дата доставки
3. Рядок 3: кількість товарів
4. Далі по одному товару в рядку:

```text
name|quantity|price
```

### Приклад
```text
оплачено
2026-04-20
2
milk|3|43.5
bread|1|29
```

## Коротко про основні методи
- `addProduct(name, quantity, price)` — додає товар або збільшує кількість існуючого
- `removeProduct(name)` — видаляє товар(и) з цією назвою
- `changeProductQuantity(name, newQuantity)` — змінює кількість, якщо `newQuantity <= 0` — видаляє товар
- `setStatus(newStatus)` — змінює статус замовлення
- `setDeliveryDate(date)` — встановлює дату доставки
- `printOrder()` — виводить замовлення в консоль
- `saveToFile(fileName)` — зберігає замовлення у файл
- `loadFromFile(fileName)` — завантажує замовлення з файлу
